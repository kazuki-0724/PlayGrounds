# Container-Bound GAS

このフォルダは、1 つのスプレッドシート内で完結するコンテナバインド GAS 実装です。

## できること

- 親スプレッドシートにカスタムメニューを追加
- モーダルダイアログから diff ファイルをアップロードし、差分リストを生成
- 同一スプレッドシート内に 差分リスト、案件マスタ、確認者マスタ、マスタ の 4 シートを用意
- 差分をファイル単位ではなく hunk 単位で 1 行ずつ展開
- 差分リストの各行で、案件と確認者をプルダウン選択
- 元 diff ファイルを同じ Drive フォルダへ保存
- 差分列から詳細表示用ページを開く

## ファイル

- appsscript.json: Apps Script マニフェスト
- Code.gs: メニュー、ダイアログ起動、diff 解析、差分リスト更新、詳細データ取得
- CreateDiffDialog.html: diff アップロード専用ダイアログ
- DiffDetail.html: hunk 単位の差分詳細表示ページ

## 使い方

1. スプレッドシートを用意し、このフォルダのファイルをコンテナバインドの Apps Script プロジェクトへ配置する。
2. スクリプトを保存し、必要なら Web アプリとしてデプロイする。
3. スプレッドシートを再読込する。
4. 差分管理 > 差分リスト作成 を実行し、diff ファイルをアップロードする。
5. 生成された 差分リスト シートで、各差分行の 案件 と 確認者 をプルダウンから選択する。
6. 必要に応じて 案件マスタ と 確認者マスタ を編集して候補を更新する。

## シート構成

- 差分リスト: hunk 単位の一覧。案件と確認者はこのシートで入力。
- 案件マスタ: 案件の候補一覧。
- 確認者マスタ: 確認者の候補一覧。
- マスタ: 元 diff 情報と差分詳細の内部保持用。通常は非表示。

## 注意点

- 差分詳細のリンクは Web アプリ URL を使います。ScriptApp.getService().getUrl() が空の場合はリンクが生成されません。
- 差分リストを再生成すると、差分リスト シートと マスタ シートの内容は置き換わります。
