# diff_maker / gas_container_bound

Google スプレッドシートにバインドして使う差分リスト作成ツールです。

diff ファイルをダイアログからアップロードすると、1 ファイルごとに以下を作成します。

- スプレッドシート内の差分リストシート 1 枚
- Google Drive 上の完成 HTML 1 ファイル
- Google Drive 上の元 diff 1 ファイル

レビュー時は、生成されたシート上で案件と確認者をプルダウン選択して運用します。

## 現在の仕様

- 実行形態はコンテナバインド型の Google Apps Script
- メニューから差分リスト作成ダイアログを開く
- 複数の diff / patch ファイルをまとめてアップロード可能
- 各 diff ファイルごとに出力名を設定可能
- 出力名はシート名と HTML ファイル名の両方に使用
- ダイアログでは side-by-side の差分プレビューを表示
- プレビューはファイル単位で折りたたみ可能
- 作成方法は 2 通り
	- 選択中を作成: 現在選択している 1 件だけ作成
	- まとめて作成: 未作成のジョブだけをまとめて作成
- ダイアログのアップロード状態はブラウザの sessionStorage に一時保持
- 既作成ジョブは 作成済み として表示
- 出力名を変更したジョブは未作成状態に戻る

## 生成されるシート

このスクリプトは、必要に応じて以下のシートを自動作成します。

### 案件マスタ

- シート名: 案件マスタ
- 1 列目に案件名を保持
- レポートシートの 案件 列のプルダウン元として使用

### 確認者マスタ

- シート名: 確認者マスタ
- 1 列目に確認者名を保持
- レポートシートの 確認者 列のプルダウン元として使用

### 出力履歴

- シート名: 出力履歴
- 生成済み出力の一覧を保持
- 同じ出力名で再作成した場合は履歴を上書き更新

列構成は以下です。

- 出力名
- シート名
- HTMLファイル
- diffファイル
- 作成日時
- 作成者
- 差分単位数
- 対象ファイル数

### 差分リストシート

アップロードした diff ファイルごとに、出力名をシート名として 1 枚作成します。

列構成は以下です。

- No
- ファイル名
- パス
- 案件
- 確認者

1 行は 1 ファイルではなく 1 差分単位です。内部的には unified diff の hunk 単位で分割して行を作成しています。

## 動作フロー

1. スプレッドシートを開く
2. メニュー 差分管理 から 差分リスト作成 を実行
3. diff / patch ファイルを 1 件以上アップロード
4. 必要に応じて各ファイルの出力名を編集
5. 以下のどちらかで作成
	 - 選択中を作成
	 - まとめて作成
6. 生成されたシートで 案件 と 確認者 を選択
7. 必要に応じて出力結果一覧のリンクから HTML を開いて確認

## Drive への出力

出力ファイルは、対象スプレッドシートが置かれている Google Drive フォルダに保存します。

- 出力名.diff
- 出力名.html

同名ファイルがすでに存在する場合は、既存ファイルをゴミ箱に移動してから新しいファイルを作成します。

## ダイアログの挙動

- 同じファイル名の diff を同一セッション中に再アップロードした場合、一覧への重複追加はしない
- 画面を閉じても sessionStorage に残っていれば次回表示時に復元される
- クリア を押すとダイアログ内のジョブ一覧と一時保存状態を削除する
- 差分プレビューは diff2html を使ってブラウザ側で生成する

## ファイル構成

- [Code.gs](Code.gs)
	- メニュー登録
	- diff 解析
	- シート生成
	- Drive 出力
	- 履歴更新
- [CreateDiffDialog.html](CreateDiffDialog.html)
	- アップロード UI
	- 差分プレビュー
	- ジョブ管理
	- 個別作成 / 一括作成
- [Preview.html](Preview.html)
	- 保存用 HTML のテンプレート

## 注意点

- 出力名に以下の予約済みシート名は使用不可
	- 案件マスタ
	- 確認者マスタ
	- 出力履歴
- 出力名の禁止文字は _ に置換する
- diff から hunk を 1 件も抽出できない場合はエラー
- レポートシートは同名シートが存在する場合、内容をクリアして再生成する

## セットアップ

1. Google スプレッドシートを作成する
2. そのスプレッドシートに紐づく Apps Script プロジェクトを開く
3. このディレクトリ内の以下 3 ファイルを配置する
	 - Code.gs
	 - CreateDiffDialog.html
	 - Preview.html
4. 必要に応じて 案件マスタ と 確認者マスタ の初期値を編集する
5. スプレッドシートを再読み込みして、メニュー 差分管理 を表示する

## 補足
